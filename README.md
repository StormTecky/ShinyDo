# Shinydo 📝

A simple Shiny app to manage your to-do list.

Developed in R, it allows you to easily add and delete tasks from a web interface.

## 🎯 Features
- Add tasks
- View the complete list
- Clear all tasks
- Lightweight Shiny interface

## 🚀 Launch the application
In an R terminal:

```r
shiny::runApp()
